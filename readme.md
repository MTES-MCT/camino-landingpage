# Camino landing page

Page provisoire d'information sur le projet Camino, accessible [ici](http://camino.beta.gouv.fr/).

---

## Npm scripts

```bash
# Install dependencies.
npm install

# Serve with hot reload at localhost:8080
npm run dev

# Build for production
npm run dist
```
